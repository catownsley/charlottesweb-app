# CharlottesWeb App

HIPAA Compliance-as-Code platform for AI-enabled health applications.

## Scope (MVP)
- Metadata-only intake (no PHI ingestion)
- HIPAA rule mapping (Security/Privacy/Breach)
- CVE/CWE exploitability-aware risk correlation
- Prioritized remediation roadmap
- Audit evidence artifact generation

## Project Structure
- `src/` application code
- `docs/tickets/` execution backlog and ticket files
- `.github/ISSUE_TEMPLATE/` ticket templates

## Getting Started
1. Create and activate a Python virtual environment
2. Add backend framework (FastAPI recommended)
3. Implement Phase 1 vertical slice: intake -> mapping -> findings -> report
